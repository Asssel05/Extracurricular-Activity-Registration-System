Extracurricular Activity App
Оқушыларға арналған үйірмелерге онлайн тіркелу жүйесі.
Бұл мобильді iOS қолданба SwiftUI көмегімен жасалған.
Қолданушылар мектептегі барлық үйірмелерді көріп, сипаттамасын оқып, орын санын тексеріп, қалаған үйірмесіне онлайн түрде тіркеле алады.
Ата-ана / оқушы мүмкіндіктері:
Үйірмелер тізімі
Үйірме сипаттамасы (орны, күні, уақыты, вместимость)
Тіркелу / тіркеуден шығу
"Менің үйірмелерім" бөлімі
Жүйеге кіру / тіркелу
Мұғалім (Admin) мүмкіндіктері:
Үйірме қосу
Қатысушылар статистикасы
